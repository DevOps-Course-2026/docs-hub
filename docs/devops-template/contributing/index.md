# 🛠️ Contributing

This section covers everything you need to create and maintain a repository
in the **DevOps Course 2026** organisation.

## What You Will Find Here

- [Template Guide](./template.md) — what the `devops-template` ships, how to
  create a repo from it, and which secrets to add.

## Quick Start

1. Go to [devops-template](https://github.com/DevOps-Course-2026/devops-template)
   and click **Use this template → Create a new repository**.
2. Add the required secrets to your new repo (see [Template Guide](./template.md)).
3. Drop your docs under `docs/` — they sync to this portal automatically on
   every push to `main`.
